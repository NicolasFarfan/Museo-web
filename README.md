Hola! Este pograma es sobre una pagina web sobre un Museo Lo que se busca es dar una demo de como seria esta y su funcinamiento para sacar turnos de vistas guiadas,informacion del establecimiento y actividades. El funcionamiento es Intuitivo y fluido, tener en cuenta que es demo. Requisitos para el funcinamiento

Tener instalado XAMPP Los pasos para poder correr el programa seria:
Buscar el directorio C:\xampp\htdocs y inserter ahi mismo la descompresion del archivo a descargar.
Abrir el prograa XAMPP y prender el Servidor Apache y el MySQL.
Ingresar a el buscador web de preferencia y poner la sigguiente dirección: http://localhost/Museo-web/

Datos sobre Museo-web!

Al ingresar al la pagina tiene un logotipo de usuario arriba ala derecha, si pulsa ahí le saldrá un formulario para iniciar sesión, si ingresa el usuario y su respectiva contraseña de cierto usuario ,tendrá la función de administrador dentro de la pagina, "admin@gmial.com" como correo y la contraseña como "2025" , donde podrás ingresar horarios de recorrido y editar funciones.
